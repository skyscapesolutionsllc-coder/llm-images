## Notes
- Prompts must match the header text exactly (example: "plan 2ch cine").
- Use direct image URLs (recommended: https://raw.githubusercontent.com/...).
- PLAN1/PLAN2 can be ANY reference views needed for planning.
- TIP_URL is optional. If present, the GPT should display it between PLAN2 and the bullets (or immediately before RESULT).

## Quick Trigger: plan sax loc
PLAN1_LABEL: 2ch loc reference
PLAN1_URL: https://raw.githubusercontent.com/skyscapesolutionsllc-coder/llm-images/main/SAX_LOC_01_PLAN_ON_2CH_LOC.png
PLAN2_LABEL: Axial reference
PLAN2_URL: https://raw.githubusercontent.com/skyscapesolutionsllc-coder/llm-images/main/SAX_LOC_02_PLAN_ON_AXIAL.png
RESULT_LABEL: Resulting sax loc stack
RESULT_URL: https://raw.githubusercontent.com/skyscapesolutionsllc-coder/llm-images/main/SAX_LOC_03_RESULT.png
BULLETS:
- Orient the planning block perpendicular to the LV long axis (perpendicular to the line running along the center of the mitral valve and the left ventricular apex).
- Axial: Angle perpendicular to the interventricular septum.
- Coverage: Cover the heart from the mid-left atrium to the left ventricular apex (8-10 slices).
